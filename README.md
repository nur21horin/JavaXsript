# Next.js + Tailwind Blog (minimal scaffold)

This workspace contains a minimal Next.js blog scaffold using Tailwind CSS.

Quick start:

```bash
# from d:\JavaSxript
npm install
npm run dev
```

Open http://localhost:3000

Notes:
- Files added: `pages/`, `components/`, `lib/posts.js`, `styles/`, `tailwind.config.js`, `postcss.config.js`, and `package.json`.
- Posts are in-memory in `lib/posts.js`. Replace with a CMS or markdown files for more posts.
